Electronic Control Module (ECM) Coding

ECM modules also known as Electronic Control Units (ECU) are dotted throughout a car. Types of ECM include Engine Control Module (ECM), Powertrain Control Module (PCM), Transmission Control Module (TCM), Brake Control Module (BCM or EBCM) and Central Control Module (CCM).

These modules perform various functions such as computing the required amount of fuel injected and when to spark the coil in order to start the engine. They collectively work together to form the on-board computer even sharing data via data lines.

Unlike other tools, the VAG Scan tool supports bidirectional communication capabilities, reading and writing to all ECMs. This means it is able to carry out actuation tests of components such as doors, windows, wipers etc to check if they are working correctly.

Code to adapt or unlock hidden features

Coding is aimed at more advanced users and workshop professionals. It allows you to modify the pre-progammed options in each ECM. It is also known as re-flashing. For example, after replacement of certain parts, the relevant ECM needs to be re-programmed in order to adapt to the new part. An example of this is, injector coding. When new injectors are replaced, the old values need to be adapted to the new values.

Many pre-progammed options can be modified and overriden; for example turning off the automatic engine start-stop system or changing the speedometer to show the speed digitally and much more.
